# Переформатирование документа по ГОСТ (Режим 3)

## Цель
Взять существующий Word-документ и применить к нему полное ГОСТ-форматирование, не меняя содержимого.

## Входные данные
1. **Существующий .docx** — файл для переформатирования

## Инструменты
- `execution/word_injector.py` — функция `reformat_existing_document()`
- `execution/gost_formatter.py` — `reformat_document()` (альтернатива)

## Пайплайн

### Шаг 1: Переформатирование
```bash
# Через word_injector
python execution/word_injector.py \
  --reformat existing.docx \
  --output formatted.docx

# Или напрямую через gost_formatter
python execution/gost_formatter.py \
  --reformat existing.docx formatted.docx
```

### Шаг 2: Проверка в Word
Открыть `formatted.docx` и убедиться:
- [ ] Шрифт: Times New Roman 14pt (таблицы — 12pt)
- [ ] Поля: лево 30мм, право 15мм, верх/низ 20мм
- [ ] Интервал: полуторный (1.5)
- [ ] Абзацный отступ: 1.25 см
- [ ] Нумерация страниц: внизу по центру
- [ ] Заголовки: H1 — 16pt, жирный, ЗАГЛАВНЫЕ; H2/H3 — 14pt, жирный
- [ ] Таблицы: границы чёрные, шрифт 12pt, по центру

## Что применяется
| Параметр | Значение |
|----------|----------|
| Шрифт | Times New Roman |
| Размер основного текста | 14pt |
| Размер таблиц | 12pt |
| Размер Heading 1 | 16pt, жирный, CAPS |
| Размер Heading 2 | 14pt, жирный |
| Размер Heading 3 | 14pt, жирный |
| Поля | 30/15/20/20 мм |
| Интервал | 1.5 |
| Абзацный отступ | 1.25 см |
| Выравнивание | По ширине (justify) |

## Краевые случаи
- **Документ без стилей** — все абзацы получают стиль Normal с TNR 14
- **Нестандартные шрифты** — заменяются на TNR, включая кириллические
- **Существующие таблицы** — границы и шрифт переформатируются
- **Изображения** — остаются без изменений
