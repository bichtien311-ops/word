# Инструкция: структура и оформление учебного пособия по ГОСТ

> Назначение: эталонная инструкция для сборки учебного пособия «Режимы нейтрали электрических сетей» в Word.
> Источник требований: ГОСТ 7.60-2003 (виды изданий), ГОСТ Р 7.0.100-2018 (библиографическое описание), ГОСТ Р 7.0.5-2008 (библиографические ссылки), ГОСТ 7.32-2017 и ГОСТ 2.105 (структура и оформление текстовых документов).
> Содержание перепроверено по открытым источникам и переработано для соответствия лицензионным ограничениям.

## 1. Что такое учебное пособие (ГОСТ 7.60-2003)

Учебное пособие — учебное издание, дополняющее или частично (полностью) заменяющее учебник, официально утверждённое в качестве данного вида издания. В отличие от учебника, может охватывать не всю дисциплину, а часть курса, и допускает авторскую подачу материала. Наше издание относится к виду «учебное пособие» для вуза.

## 2. Структура издания (порядок элементов)

Обязательные элементы — по порядку следования:

1. Титульный лист (лицевая сторона).
2. Оборот титульного листа: индексы УДК и ББК, авторский знак, сведения о рецензентах, аннотация, гриф, выходные данные, ISBN (если есть).
3. Оглавление (содержание) — автособираемое, с указанием страниц.
4. Список условных обозначений, сокращений и терминов (если их много) — опционально.
5. Введение (предисловие): актуальность темы, цель и задачи пособия, для кого предназначено, связь с дисциплиной, краткий обзор структуры.
6. Основная часть: разделы (модули) → главы (лекции) → параграфы. В конце каждой главы — контрольные вопросы и (при необходимости) примеры/задачи.
7. Заключение: основные выводы, область применения материала.
8. Список использованных источников / литературы (по ГОСТ Р 7.0.100-2018).
9. Приложения (справочные таблицы, крупные схемы, нормативные выдержки) — опционально.

## 3. Оформление основного текста (соответствует gost_formatter)

| Параметр | Значение |
|----------|----------|
| Шрифт | Times New Roman |
| Размер основного текста | 14 пт |
| Размер текста в таблицах | 12 пт |
| Межстрочный интервал | полуторный (1,5) |
| Абзацный отступ | 1,25 см |
| Выравнивание текста | по ширине |
| Поля (лево/право/верх/низ) | 30 / 15 / 20 / 20 мм |
| Нумерация страниц | внизу по центру, сквозная |
| Заголовок 1 уровня (раздел/модуль) | 16 пт, полужирный, ПРОПИСНЫЕ |
| Заголовок 2–3 уровня (глава/параграф) | 14 пт, полужирный |

## 4. Нумерация и рубрикация

- Разделы (модули) нумеруются арабскими цифрами: 1, 2, 3…
- Главы (лекции) — внутри модуля: 1.1, 1.2…; параграфы — 1.1.1 и т.д.
- Каждый крупный раздел желательно начинать с новой страницы.
- Заголовки без точки в конце, не подчёркиваются.

## 5. Формулы

- Формулы выносятся отдельной строкой, выравниваются по центру, нумеруются справа в круглых скобках сквозно либо по разделам: (1), (2) или (3.1), (3.2).
- После формулы — расшифровка обозначений со слова «где», каждый символ с новой строки.
- Десятичный разделитель — запятая (32,09, а не 32.09).
- Все обозначения русскоязычные/общепринятые технические. Исходные формулы берутся 1:1, изменяются только подставляемые числа.

## 6. Таблицы и рисунки

- Таблицы: над таблицей справа «Таблица N — Название». Шрифт 12 пт. Границы чёрные.
- Рисунки: подпись под рисунком по центру «Рисунок N — Название».
- Нумерация сквозная или по разделам, единообразно во всём пособии.
- На каждую таблицу и рисунок должна быть ссылка в тексте.

## 7. Список литературы (ГОСТ Р 7.0.100-2018)

- Источники приводятся в порядке появления ссылок в тексте либо по алфавиту.
- Ссылки в тексте — в квадратных скобках [12] или [12, с. 34].
- Каждая запись содержит автора, заглавие, сведения об издании, место, издательство, год, объём.
- Нормативные документы (ПУЭ, ПТЭ, ГОСТ) и учебники оформляются раздельно по единому стандарту.

## 8. Контрольные вопросы

В конце каждой лекции — блок «Контрольные вопросы» (5–8 пунктов), проверяющий ключевые формулы, алгоритмы и нормы из лекции. Это требование уровня методической литературы и его стоит выдержать для всех 17 лекций.
